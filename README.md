# 🧮 Scientific Calculator

A modern and responsive **Scientific Calculator** built using **HTML, CSS, and JavaScript**. This calculator supports both basic arithmetic and advanced scientific operations, including trigonometric functions, logarithms, powers, square roots, and mathematical constants. It also features **RAD/DEG mode switching**, **calculation history**, and a **Dark Mode** for an enhanced user experience.

## ✨ Features

- Addition (`+`)

- Sine (`sin`)

- Light/Dark Theme Toggle

## 📐 Angle Modes

### RAD Mode
Trigonometric functions operate using radians.

### DEG Mode
Trigonometric functions automatically convert degrees into radians before calculation.

Examples:

```text
sin(90) = 1      (DEG Mode)
sin(π/2) = 1     (RAD Mode)
```
